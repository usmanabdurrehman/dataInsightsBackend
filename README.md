## How to run

- git clone https://github.com/usmanabdurrehman/dataInsightsBackend.git
- cd dataInsightsBackend
- pip install virtualenv

#### For Linux

- virtualenv myenv
- source myenv/bin/activate
- pip install -r requirements
- python app.py

#### For Windows

- python -m venv myenv
- myenv\Scripts\activate
- pip install -r requirements
- python app.py
